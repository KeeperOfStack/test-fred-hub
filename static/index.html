ⓘ About Test Fred Hub

What is this?

A self-hosted hub for managing the test-fred Minecraft server. Built with FastAPI + vanilla JS, inspired by the Mojang Launcher.

Plugin Sources

  • modrinth — Modrinth REST API, SHA-512 hash matching
  • hangar — Hangar (PaperMC) projects API
  • geyser — official download.geysermc.org/v2
  • spiget — Spiget mirror of SpigotMC resources

Update Workflow

  1. Hub downloads jar to plugins/update/ with matching filename
  2. You click Restart & Apply
  3. Paper detects the update folder and swaps jars atomically on startup

Permissions

The hub talks to /var/run/docker.sock directly — no sudo needed because kratos is in the docker group. Restart/start/stop all flow through Docker's API.
